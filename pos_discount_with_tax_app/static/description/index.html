POS Fixed/Percentage Discount on Taxed/Untaxed Amount

Using this module you can apply fixed/percentage discount on whole pos order or orderline on taxed or untaxed amount.

Configuration

Configure on what amount you want to add discount (taxed or untaxed).

Add Discount on Order

You can add fixed or percentage discount on order.

Add Discount on Orderline

You can add fixed or percentage discount on orderline.

Discount in Receipt

You can see total discount on (order/orderline) in receipt.

Discount in Invoice

You can see discount on (order/orderline) and discount type (fixed/percentage) in invoice.

Balanced Journal Entries

Journal entry for discount is also managed.
